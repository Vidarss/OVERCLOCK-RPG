# OVERCLOCK - Scenario/Background Prompts

## Base Prompt Template

```
Create a pixel art BACKGROUND SCENARIO for a cyberpunk idle RPG called "OVERCLOCK".

Zone: [ZONE_NAME]
Tier: [1-6]

=== STYLE RULES ===
- 32-bit retro pixel art, clean edges, NO anti-aliasing
- Size: 1920x1080 or 16:9 aspect ratio
- Dark cyberpunk aesthetic - neon accents on dark base
- NO characters, enemies, or UI elements
- Parallax-ready layers preferred (foreground, mid, background)
- Atmospheric, moody, hostile environment
- Should feel like enemy territory - dangerous, unwelcoming

=== TIER-SPECIFIC RULES ===
[See tier characteristics below]
```

---

## Tier Characteristics

### === TIER 1 ===
Theme: Outer digital perimeter, corrupted network edges
Colors: Dark blues, faint cyan/green glows
Detail: Simple structures, basic corruption
Mood: Uneasy, entry point to hostile territory

### === TIER 2 ===
Theme: Firewall zones, security barriers, checkpoint areas
Colors: Orange/red warnings, darker base
Detail: More complex structures, visible defenses
Mood: Guarded, actively hostile

### === TIER 3 ===
Theme: Deep system kernel, industrial processing cores
Colors: Deep purples, harsh whites, critical reds
Detail: Dense machinery, data streams, complex architecture
Mood: Cold, mechanical, efficient danger

### === TIER 4 ===
Theme: Core system heart, ancient code repositories
Colors: Black voids, intense neon accents
Detail: Massive structures, reality distortions
Mood: Oppressive, overwhelming power

### === TIER 5 ===
Theme: The void between systems, corrupted space
Colors: Pure blacks, unstable color shifts
Detail: Reality breaking, impossible geometry
Mood: Cosmic dread, beyond comprehension

### === TIER 6 ===
Theme: End of all systems, apocalyptic collapse
Colors: All colors bleeding, void consuming
Detail: Maximum chaos, total corruption
Mood: Finality, the end of digital existence
