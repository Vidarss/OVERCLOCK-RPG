# Monster Sprite Generation Prompts

Use these prompts with AI image generators (Midjourney, DALL-E, Leonardo, etc.)

---

## BASE PROMPT TEMPLATE

```
Create a pixel art enemy sprite for a cyberpunk idle RPG called "OVERCLOCK".

Enemy name: [NAME]
Tier: [NUMBER] - "[TIER NAME]"

=== TIER CHARACTERISTICS ===
[TIER DESCRIPTION]

=== STYLE ===
- 32-bit retro pixel art, clean edges, NO anti-aliasing
- Size: 256x256 pixels
- Background: SOLID SINGLE COLOR (dark purple #1a0a2e or dark blue #0a0a2e)
- Palette: 2-3 neon accent colors on dark base

=== DESIGN RULES ===
- Interpret name LITERALLY or ABSURDLY
- Add ironic/funny elements (tiny legs, sad face, corporate accessories)
- FUN BUT SCARY - make players smile but feel uneasy

FORBIDDEN:
- NO text, numbers, or symbols ON sprite body
- NO UI elements, health bars, labels, watermarks
- Background MUST be SOLID SINGLE COLOR
- PURE CHARACTER ONLY
```

## FILE NAMING & STORAGE

Save sprites to:
```
public/sprites/enemies/
├── tier1/
│   ├── malware_bat.png
│   ├── corrupt_proc.png
│   └── ...
├── tier2/
│   └── ...
└── tier10/
    └── ...
```

Update `ENEMY_CONFIG.spriteMap` in `game.config.ts` to map enemy names to sprite paths.
