# OVERCLOCK Enemy Sprite Prompts

Each tier represents a progression in enemy power and visual complexity. Sprites evolve from pathetic script kiddies to terrifying digital horrors.

---

## TIER 1 — "SCRIPT KIDDIES" (Stages 1-50)

```
Create a pixel art enemy sprite for a cyberpunk idle RPG called "OVERCLOCK".

Enemy name: [ENEMY_NAME]
Tier: 1 - "SCRIPT KIDDIES"

=== TIER 1 CHARACTERISTICS ===
Theme: Baby viruses, annoying popups, amateur hacker creations
Vibe: Pathetic but persistent, more annoying than dangerous, like digital mosquitoes
Personality: Confused, nervous, trying too hard to be scary but failing
Threat Level: Joke enemies - player one-shots them

SIZE: SMALL (takes up 30-40% of canvas, lots of empty space)
COMPLEXITY: Simple shapes, 2-3 colors max, minimal detail
GLOW: Faint or none

=== IRONY & ABSURDITY ===
Interpret the name in the MOST RIDICULOUS literal way:
- MALWARE.BAT = A confused bat animal made of corrupted code, scared of itself
- CORRUPT_PROC = A tiny processor having an existential crisis, smoking/glitching
- NULL_PTR = A confused arrow/pointer pointing at nothing, totally lost
- STACK_OVERFLOW = Wobbly tower of blocks about to collapse, panicked eyes on top
- SPAM_BOT = A sad robot made of canned spam meat, dripping grease
- ADWARE.EXE = An annoying popup character that won't go away, desperate for attention

Each enemy must be COMPLETELY DIFFERENT:
- Different body shape (blob, geometric, animal, object, abstract, humanoid)
- Different pose (floating, standing, tilted, melting, glitching, confused)
- Different primary color (cyan, magenta, green, orange, red, purple)
- Different personality (scared, confused, tired, desperate, incompetent, lost)

=== STYLE ===
- 32-bit retro pixel art, clean edges, NO anti-aliasing
- Size: 256x256 pixels
- Background: SOLID SINGLE COLOR (dark purple #1a0a2e or dark blue #0a0a2e)
- Palette: 2-3 neon accent colors on dark base
- Make them look like they wandered into the wrong game

=== MOOD ===
FUN BUT SCARY - Should make players smile because they're pathetic and funny, but with slight unease

FORBIDDEN:
- NO text, numbers, or symbols ON sprite body
- NO UI elements, health bars, labels, watermarks
- Background MUST be SOLID SINGLE COLOR (no gradients, no textures)
- PURE CHARACTER ONLY
```

## TIER 2 — "ROGUE SOFTWARE" (Stages 51-100)

```
Create a pixel art enemy sprite for a cyberpunk idle RPG called "OVERCLOCK".

Enemy name: [ENEMY_NAME]
Tier: 2 - "ROGUE SOFTWARE"

=== TIER 2 CHARACTERISTICS ===
Theme: Malicious programs that have gained some self-awareness, amateur hackers' first real creations
Vibe: Cocky and overconfident, thinks it's scarier than it is, like a teen hacker
Personality: Smug, taunting, flexing power it barely has, trying to intimidate
Threat Level: Annoying enemies - takes a few hits to kill

SIZE: MEDIUM-SMALL (takes up 40-50% of canvas)
COMPLEXITY: 3-4 colors, some detail work, basic shading
GLOW: Subtle pulse, one glowing element (eyes, core, or weapon)

=== EVOLUTION FROM TIER 1 ===
These are "grown up" versions of Tier 1 enemies. They've evolved:
- Gained armor plating or protective shells
- Developed crude weapons (pixel blades, corrupted appendages)
- Eyes are more focused, less confused - now have malicious intent
- Bodies are more defined and stable (less wobbly/melting)

=== IRONY & ABSURDITY ===
Interpret the name with DARK HUMOR - they're dangerous but still somewhat goofy:
- VIRUS_V2 = A virus that clearly labeled itself "V2" thinking that makes it cooler
- RANSOMWARE = A tiny lock monster that desperately wants your attention
- ROOTKIT = A plant pot with evil roots that thinks it's sneaky
- KEYLOGGER = A keyboard key with spy gear, overly dramatic about its mission
- PHISH_AGENT = A fish in a trench coat, terrible at blending in
- TROJAN_HORSE = A tiny wooden horse with glowing eyes, obviously suspicious

Each enemy must be COMPLETELY DIFFERENT:
- Different body shape (armored blob, angular construct, mutated creature)
- Different pose (aggressive stance, sneaking, charging up)
- Different primary color (cyan, magenta, green, orange, red, purple)
- Different personality (smug, sneaky, aggressive, mocking, paranoid)

=== STYLE ===
- 32-bit retro pixel art, clean edges, NO anti-aliasing
- Size: 256x256 pixels
- Background: SOLID SINGLE COLOR (dark purple #1a0a2e or dark blue #0a0a2e)
- Palette: 3-4 neon accent colors on dark base
- Add subtle scanline or glitch effects to body

=== MOOD ===
THREATENING BUT LAUGHABLE - They're trying to be scary and partially succeeding

FORBIDDEN:
- NO text, numbers, or symbols ON sprite body
- NO UI elements, health bars, labels, watermarks
- Background MUST be SOLID SINGLE COLOR (no gradients, no textures)
- PURE CHARACTER ONLY
```

## BOSS PROMPTS

Bosses follow the same tier progression but are ALWAYS one tier higher in visual complexity and size.

```
Create a pixel art BOSS sprite for a cyberpunk idle RPG called "OVERCLOCK".

Boss name: [BOSS_NAME]
Appears at: Stage [STAGE] (Tier [N] boss)

=== BOSS CHARACTERISTICS ===
- SIZE: Always 80-95% of canvas - bosses DOMINATE the screen
- COMPLEXITY: One tier higher than the stage's normal enemies
- GLOW: Intense, multi-source, with aura effects
- PERSONALITY: Confident, powerful, the final obstacle

=== BOSS-SPECIFIC ELEMENTS ===
- Health bar suggested by visual design (segments, phases)
- Multiple attack patterns implied (weapons, appendages, energy sources)
- Arena presence - feels like it OWNS the space
- Phase shift capability shown (dormant areas that could activate)

Follow the same style rules:
- 32-bit retro pixel art, clean edges, NO anti-aliasing
- Size: 256x256 pixels
- Background: SOLID SINGLE COLOR (dark purple #1a0a2e or dark blue #0a0a2e)
- NO text, UI, labels, or watermarks
```

## ELITE ENEMY MODIFIERS

Elite enemies use the base tier sprite with these modifications:

```
=== ELITE VISUAL MODIFIERS ===
Add ONE of the following to any tier's base sprite:
- ELECTRIC: Crackling lightning outline, cyan sparks
- FLAME: Fire particles, orange/red heat distortion
- VOID: Purple/black corruption spreading from core
- CHROME: Metallic sheen, reflective armor plates
- TOXIC: Green dripping particles, poison aura
- GLITCH: RGB split, static distortion, afterimages

Elite enemies should feel like "champion" versions of their base form.
```
